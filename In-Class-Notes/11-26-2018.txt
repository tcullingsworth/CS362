Storing data:
INSERT
UPDATE
DELETE

INSERT values - insert values into a table
INSERT SELECT - insert values into existing table using data from another table
SELECT INTO   - insert values into new table using data from another table

UPDATE records
UPDATE table SET values - will update every record in the table with the values you specify
UPDATE table SET values WHERE criteria - this will only update the record or records that match the criteria

DELETE FROM table - delete all records from a table
TRUNCATE table - will also delete all records from table

DELETE FROM table WHERE criteria - will only delete records from table that match criteria

use OUTPUT to record deleted or inserted data
